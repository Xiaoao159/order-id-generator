## Description
Unique order id generator for Firebase

### Usage

```js
import orderId from 'order-id-generator';

const uid = 'x84zLYuvLHhlpoHbR93vytP0wsVU';
const id = orderId(uid);
// j4jqp04zryllvyld
```
